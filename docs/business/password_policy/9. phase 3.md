## Các vấn đề cần rà soát trước triển khai

| Nội dung | Chi tiết |
|---|---|
| DB chưa xác định | Shortlink/Deeplink, Invoice Kế toán |
| Multi-DB service | CDP, Fraud Management, SSO, SignServer |
| Hệ thống outsource | HR Online, EOffice, Phần mềm Kế toán |
| MongoDB Cluster | Voucher System, CDP, SignServer |
| Shared DB/IP | Một số dịch vụ dùng chung DB/IP cần kiểm tra ownership |
| Legacy application | Oracle/SQLServer cũ cần test compatibility |

## Checklist triển khai chuẩn

| Hạng mục | Nội dung |
|---|---|
| Password Policy | Tạo profile/rule theo chuẩn bảo mật |
| User Service | Tạo user mới theo naming convention |
| Audit | Enable audit login/action |
| Expired User | Expire user cũ sau chuyển đổi |
| Rollback | Chuẩn bị rollback plan |
| Hardcode Password | Rà soát ứng dụng |
| Session Review | Kiểm tra session tồn đọng |
| Monitoring | Theo dõi sau triển khai |
