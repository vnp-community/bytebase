## Kết quả mong đợi

| Nội dung | Mục tiêu |
|---|---|
| Password Policy | Áp dụng thống nhất cho toàn bộ User Database |
| Audit | Kiểm soát truy vết đầy đủ các user DB |
| User Service | Chuyển đổi sang user mới theo chuẩn bảo mật |
| User cũ | Được expired và theo dõi trước khi thu hồi hoàn toàn |
| Quy trình | Chuẩn hóa quy trình cấp phát/quản lý DB User |
| Compliance | Đáp ứng yêu cầu PCI-DSS, ISO27001 và audit nội bộ |

## Các hạng mục cần tiếp tục theo dõi

- Công cụ thông báo tự động khi mật khẩu sắp hết hạn
- Rà soát ứng dụng hardcode DB password
- Theo dõi session/kết nối sử dụng user cũ
- Hoàn thiện checklist rollback khi đổi user ứng dụng
- Đồng bộ quy trình triển khai với GSVH/ANSP/QLTT
- Mở rộng triển khai cho các dịch vụ Production còn lại
