## Tổng hợp theo hệ quản trị Database

| Database | Số lượng dịch vụ | Ghi chú triển khai |
|---|---|---|
| Oracle | 13+ | Ưu tiên triển khai profile/password expiration |
| MongoDB | 5+ | Chuẩn hóa service account và audit auth |
| PostgreSQL | 8+ | Chuẩn hóa role/password policy |
| SQLServer | 1 | Áp dụng CHECK_POLICY/CHECK_EXPIRATION |
| MySQL | 1 | Kiểm tra khả năng enforce password policy |

## Checklist triển khai cho từng dịch vụ

| Hạng mục | Nội dung |
|---|---|
| Password Policy | Tạo profile/rule theo chuẩn |
| User Service mới | Tạo mới theo naming convention |
| Audit | Enable login/action audit |
| Hardcode password | Rà soát ứng dụng |
| Rollback | Chuẩn bị rollback plan |
| Monitoring | Kiểm tra monitoring sau chuyển đổi |
| Session cũ | Rà soát session tồn đọng |
| Expired User cũ | Thực hiện sau khi ổn định |
