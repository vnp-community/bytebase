# Module: Settings Pages

## Scope
Workspace-level settings: members, users, groups, roles, IDPs, environments,
instances, SQL review, subscription, audit log, general settings, MCP, service accounts.

## Primary Files
- `MembersPage.tsx` — workspace member management (invite, remove, role assign)
- `UsersPage.tsx` — user CRUD
- `GroupsPage.tsx` — group management
- `RolesPage.tsx` — custom role management
- `EnvironmentsPage.tsx` — environment ordering and reorder
- `InstancesPage.tsx` — instance list + sync
- `CreateInstancePage.tsx` — new instance wizard
- `InstanceDetailPage.tsx` — single instance detail
- `IDPsPage.tsx` — identity provider list
- `IDPDetailPage.tsx` — IDP configuration
- `GeneralPage.tsx` — workspace name/logo/branding
- `SQLReviewPage.tsx` — SQL review config list
- `SQLReviewDetailPage.tsx` — SQL review rule configuration
- `SQLReviewCreatePage.tsx` — create new SQL review config
- `SubscriptionPage.tsx` — license management
- `AuditLogPage.tsx` — audit log viewer
- `ProfilePage.tsx` — user profile edit
- `TwoFactorSetupPage.tsx` — 2FA configuration
- `CustomApprovalPage.tsx` — approval flow config
- `RiskAssessmentPage.tsx` — risk assessment rules
- `DataClassificationPage.tsx` — data classification config
- `GlobalMaskingPage.tsx` — global masking rules
- `SemanticTypesPage.tsx` — semantic type definitions
- `ServiceAccountsPage.tsx` — service account management
- `WorkloadIdentitiesPage.tsx` — workload identity management
- `MCPPage.tsx` — MCP server configuration
- `IMPage.tsx` — IM integration (Slack, Teams, etc.)

## Dependencies
- **API Clients**: `userServiceClientConnect`, `instanceServiceClientConnect`, `settingServiceClientConnect`, `roleServiceClientConnect`, `groupServiceClientConnect`, `identityProviderServiceClientConnect`, `reviewConfigServiceClientConnect`, `auditLogServiceClientConnect`, `subscriptionServiceClientConnect`
- **Stores**: `useAuthStore` (permission checks), `useSubscriptionV1Store` (feature gating)
- **Permissions**: workspace-level `bb.users.*`, `bb.instances.*`, `bb.settings.*`, `bb.roles.*`, `bb.groups.*`

## Common Tasks → File to Edit
| Task | File |
|---|---|
| Add member filter | `membersPageActions.ts` or `MembersPage.tsx` |
| Add instance column | `InstancesPage.tsx` |
| Add IDP field | `IDPDetailPage.tsx` |
| Modify general setting | `general/` subdirectory |
| Add SQL review rule | `SQLReviewDetailPage.tsx` |
| Edit user profile | `profile/` subdirectory |
| Change 2FA flow | `two-factor/` subdirectory |

## Prohibited
- No `useVueState` calls — these pages are fully React
- No raw `z-index` — use overlay system
- No `fetch('/v1/...')` — use ConnectRPC clients
- Use semantic color tokens, not raw Tailwind colors
