# Module: Shared React Components

## Scope
Reusable UI components shared across all React pages. Check here before creating
a new component — it may already exist.

### UI Primitives (`ui/` subdirectory)
Shadcn-style component library. Use these instead of building custom:
- Button, Dialog, Sheet, AlertDialog, DropdownMenu, Select, Tabs, Table,
  Tooltip, Input, Checkbox, Badge, Card, Separator, ScrollArea, etc.

## Prohibited
- DO NOT create a new component if one already exists in this list
- DO NOT create inline styled components — use Tailwind + semantic tokens
- DO NOT use `document.body` for portals — use overlay system from `src/react/lib/overlay.ts`
- All components must use named exports (not `export default`)
